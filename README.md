# Power BI Consolidation Exercise - Netflix Dashboard

## Descripción del Proyecto

Este proyecto es un ejercicio de consolidación de Power BI que consiste en crear un dashboard interactivo utilizando datos de Netflix. El objetivo es aplicar las habilidades de visualización de datos, transformación de datos y diseño de reportes en Power BI.

## Estructura de Carpetas

```
PowerBI/
├── Data/
│   └── Netflix.xlsx          # Dataset principal con información de Netflix
├── Desing/
│   ├── Netflix_Background.jpg
│   ├── Netflix_backgroun_slide.png
│   ├── Netflix_logo22966877073590197.png
│   ├── Disney_logo.png
│   ├── HoC.jpg               # House of Cards
│   ├── strangers.jpg         # Stranger Things
│   └── Theme.json            # Tema personalizado de colores
├── Consolidation Exercise ES.pdf  # Instrucciones del ejercicio
├── PC2_Bi_Grupo_1.pbix       # Archivo del proyecto Power BI
└── PC2 Consolidation Exercise Power BI.zip
```

## Objetivos del Ejercicio

1. **Importar y transformar datos** desde el archivo Excel de Netflix
2. **Crear visualizaciones** interactivas y significativas
3. **Aplicar diseño profesional** utilizando el tema y recursos gráficos proporcionados
4. **Implementar filtros y segmentadores** para análisis dinámico
5. **Consolidar conocimientos** de Power BI Desktop

## Datos Disponibles

El dataset `Netflix.xlsx` contiene información sobre:

- Catálogo de contenido (películas y series)
- Información de títulos
- Fechas de lanzamiento
- Géneros y categorías
- Países de origen

## Recursos de Diseño

### Tema de Colores (Theme.json)

El tema "High Contrast" incluye una paleta de colores profesional:

- **Rojo Netflix**: `#B4131D`
- **Azul oscuro**: `#002050`
- **Gris claro**: `#C2BCBC`
- **Fondo**: `#FFFFFF`
- **Texto**: `#070f25`

### Imágenes Disponibles

- Logos de Netflix
- Fondos para slides
- Imágenes de series populares (House of Cards, Stranger Things)

## Cómo Usar

1. Abrir `PC2_Bi_Grupo_1.pbix` en Power BI Desktop
2. Verificar la conexión con el archivo `Data/Netflix.xlsx`
3. Aplicar el tema desde `Desing/Theme.json` (Vista → Temas → Buscar temas)
4. Utilizar las imágenes de la carpeta `Desing/` para personalizar el dashboard

Consultar el archivo `Consolidation Exercise ES.pdf` para las instrucciones completas del ejercicio.
